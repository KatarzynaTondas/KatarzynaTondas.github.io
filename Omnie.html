O mnie

➤ Praca z danymi:

Praca z danymi była w kręgu moich zainteresowań już od czasów studiów w dziedzinie geodezji i kartografii, podczas których poznałam zasady oraz metody przedstawiania danych na mapach, specyfikę wykonywania analiz przestrzennych, pracę z zobazowaniami fotogrametrycznymi oraz danymi pozyskanymi metodami teledetekcyjnymi. Obecnie specjalizuje się w analizie dużych zbiorów danych za pomocą jęzeka SQL oraz Python. Skutecznie współpracuję z zespołami projektowymi z zakresu finansów, kontrolingu oraz spraw osobowych, implementuję rozwiązania BI orazwykorzystuję narzędzia do wizualizacji danych. Mój cel to nie tylko precyzja i dokładność wanalizie danych, ale przede wszystkim tworzenie realnej wartości dla organizacji poprzezodpowiednią pracę z danymi.

➤ Wykorzystywane technologie:

✧ Python

✧ SQL - praca w MS SQL oraz Oracle (CTE, DML, DDL), SQL Server

✧ DWH - znajomość specyfiki hurtowni danych

✧ Microsoft Power BI - DAX, Power Query, RLS oraz narzędzia DAX Studio, Tabular Editor, Bravo for Power BI, Measure Killer

✧ SSIS - SQL Server Integration Services

✧ SSRS - SQL ServerReporting Services

✧ SSAS - SQL Server Analysis Services

✧ MS Excel - tabele przestawne, zaawansowane funkcje, zagnieżdżanie funkcji, VBA for Excel

✧ Tableau - praca w Tableau Desktop, Tableau Prep, zarządzanie Tableau Server

✧ ArcGIS - analizy przestrzenne, klasyfikacja danych teledetekcyjnych, wizualizacja danych na mapach

✧ QGis - analizy przestrzenne, wizualizacja danych, korzystanie z wtyczek

✧ Canva - projektowanie kart raportów
