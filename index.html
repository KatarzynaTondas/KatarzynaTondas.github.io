Analysis of Sales
Tableau Project

The main purpose of the created report is to display sales data for the desired period. Thanks to interactive data filtering, the report viewer can determine which period should be analyzed, as well as which Region / Country should be presented. A key aspect of the created visualization helpful especially to managers or salesmen is the listing of Customers who generate the most sales. The report was created based on data provided by Tableau Software.
